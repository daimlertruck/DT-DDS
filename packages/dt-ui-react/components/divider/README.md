# Divider

This component defines a divider between components

## Usage

```jsx
import { Divider } from '@dt-ui/react';

export const App = () => {
  return <Divider />;
};
```

## API

### Divider

| Property | Type                  | Default | Description                                 |
| -------- | --------------------- | ------- | ------------------------------------------- |
| `style`  | `React.CSSProperties` | -       | CSS properties to be applied to the Divider |
